# Using
## Images folder
* every image has unique name
* images folders structure
    * <Album #1 name>
        * <Image #1>
        * <Image #2>
    * <Album #2 name>
        * <Image #3>
        * <Image #4>
        * <...>
## Setup server
* checkout server
* configure .env
    * set GALLERY_PATH wirh your images folder path
    * build and start server
```bash
cd server; docker-compose build; docker-compose up -d
```
